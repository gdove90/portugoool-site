# 12 · Logos

Final logo files per variant — see `00_brand/logo-guidelines.md` for rules. `light/` (for paper bg), `dark/` (for ink bg), `gold/` (premium contexts), `svg/` (master vectors — source of truth for all raster exports), `badge/` (brand badges, one subfolder per badge: `goool-crest-v1/` GOOOL soccer-ball crest, `gol-trophy-v1/` GOL trophy crest — each holds artwork + print/embroidery spec sheets).
