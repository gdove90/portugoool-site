# GOOOL Athletics — Sub-brand Marks

Lettering and marks for the GOOOL Athletics line (performance apparel).

| File | What it is | Use |
|---|---|---|
| `GA-MONOGRAM-LETTERING-v1.png` | Interlocked GA monogram, solid black on white, 1254x1254 | **Canonical lettering/font reference for building Athletics crests.** The italic geometric letterform style here is the sub-brand's type voice — new crests and lockups should match this cut. |

Style notes (from the reference):
- Heavy geometric slab strokes, forward italic slant
- G and A share/interlock strokes — the A's crossbar reads as part of the G
- Squared terminals, tight rounded outer corners
- Always monochrome: black on light, cream/silver-white (#E6E7E8) on dark garments
