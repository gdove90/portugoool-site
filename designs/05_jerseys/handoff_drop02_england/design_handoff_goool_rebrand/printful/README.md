## Printful product
Use **All-Over Print Recycled Unisex Sports Jersey** (or the current AOP
soccer-style jersey in the catalog). In the design tool:
1. Upload the FRONT file to the front panel, scale to full bleed (edge to edge).
2. Upload the BACK file to the back panel, full bleed.
3. Sleeves/collar: pick the solid body color per jersey (hex below) or
   reuse the FRONT file zoomed to a solid area.
4. Order ONE draft sample per colorway before publishing (sublimation
   colors shift; check the gold #C9A227 especially on the Blackout).

## Notes
- **Hem hits:** Printful's back print zone doesn't reach the actual hem
  stitching — bottom-align the hem file in the back zone for the lowest
  legal placement, or use the "outside label" placement if the product
  supports it.
- **T3 Radiating:** true off-the-seam bleed requires an all-over-print
  product; on DTG the arcs run to the print-area edge instead. If you want
  the full bleed, list T3 on the AOP tee and I'll re-export panel files.
- **Colors:** ≤4 solid colors per graphic, no gradients — DTG-safe as-is.
- Order one sample per design; check gold #C9A227 opacity on black garments.
