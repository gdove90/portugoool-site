# 06 · Tshirts

Casual shirt designs (Brush Script, GOOOOOOOL!!!, Vamos, Lisbon, Match Day, We Don't Whisper Goals + future). Same flow as jerseys: `concepts/` → `finals/` → `tech-packs/`.


## Signature detail (standard on all casual tees, adopted 2026-07-08)
**"MADE FOR THE MOMENT."** — back neck, just below the collar seam.
Anton, uppercase, tracked out (~0.3em), small (≈7cm wide print), single
color: gold on dark bodies, ink on light bodies. One line, with period.
This is the line customers find when they look closer — like a hem tag,
not a billboard. Front graphics never include it; it's the constant while
front graphics change per design.
