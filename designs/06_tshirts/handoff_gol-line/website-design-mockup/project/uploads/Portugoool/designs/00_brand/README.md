# 00_brand — Brand Foundation

The single source of truth for PORTUGOOOL's identity. Every design decision
in this repo — site UI, apparel, packaging, social, email — should trace back
to a rule in this folder.

| File | What it defines |
|---|---|
| [brand-guide.md](brand-guide.md) | Mission, vision, core emotion, story, philosophy |
| [colors.md](colors.md) | Palette with HEX/RGB and usage rules |
| [typography.md](typography.md) | Font stacks, scale, responsive rules |
| [logo-guidelines.md](logo-guidelines.md) | Wordmark usage, clearspace, misuse |
| [voice.md](voice.md) | Tone across homepage, product, email, social |
| [materials.md](materials.md) | Fabric specs and printing methods |
| [design-principles.md](design-principles.md) | The non-negotiables |

**Rule for future sessions:** if a visual decision isn't documented here,
document it here first, then implement it.
